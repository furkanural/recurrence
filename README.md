# Tekrarlama ilişkisi ve F(0) değeri verilen fonksiyonun istenen adımdaki değerinin hesaplanması

#### F(0) = 5
#### F(1) = (F(0) + 3) *2 = 16
#### F(2) = (F(1) + 3) *2 = 38
####...
#### F(n+1) = (F(n) + 3) *2

## --recurrence fonksiyon ilk_değer adım_sayısı
### > recurrence "+ 3 * 2" 5 0
### 5

### > recurrence "+ 3 * 2" 5 1
### 16

### > recurrence "+ 3 * 2" 5 2
### 38

### > recurrence "+ 3 * 2" 5 20
### 11534330
